# NHC Cash Offer Hub

A unified React + Vite app consolidating mailing tracker, flip tracking, hold income, cash offer calculator, and package deal analysis for New Home Collective / BE Property Ventures.

## Stack
- React + Vite
- Supabase (shared NHC project)
- Vercel

## Pages
- **Dashboard** — pipeline summary
- **Mailing Tracker** — Vyral campaigns + deal sourcing
- **Flips** — buy/rehab/sell properties with P&L
- **Holds** — rental properties with monthly income tracking
- **Cash Offer Calculator** — 3-option proposal generator (ported from calculator.html)
- **Package Deals** — multi-property bundle analysis
